Donameet
=========

> Connecting donors with those in need

## How to Install
```bash
# Setup virtualenv
python3 -m venv env
source env/bin/activate

# Install
pip install -r requirements.txt

# Run locally
python serve.py
```

## Technology :rocket:
- Python 3
- [Flask](http://flask.pocoo.org/)
- [SQLAlchemy](http://www.sqlalchemy.org/)
